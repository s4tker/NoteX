# NoteX

**NoteX** is a Notion-like web notes application built using only **HTML, CSS, and pure JavaScript**, without frameworks or external libraries.

Its goal is to work as a **digital brain**, allowing users to organize notes into folders, write in Markdown, and keep all information stored locally and privately.

**Live Demo (GitHub Pages):**  
[NoteX](https://s4tker.github.io/NoteX/)

---

##  Features

-  Folder and note system
-  Editor with **Markdown** support
-  **Edit / Preview** mode
-  Real-time search
-  Persistence using **LocalStorage**
-  Light / Dark mode
-  Drag & Drop to move notes
-  Clean architecture with centralized state
-  No backend, no dependencies, no tracking
